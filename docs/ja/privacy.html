プライバシーポリシー

Luminessa は、お客様の写真、動画、個人情報を当社のサーバーに収集、保存、送信しません。カメラプレビュー、LUT のレンダリング、撮影、書き出しはすべてお客様のデバイス上でローカルに処理されます。

アプリの設定をデバイス上でローカルに保存するために、UserDefaults などの標準的な Apple API を使用しています。

サブスクリプション

Luminessa Pro を購入すると、Apple が支払いを処理します。RevenueCat がサブスクリプションの状態を確認し、Luminessa が Pro 機能をアンロックできるようにします。

お客様の写真、動画、カメラのフレーム、位置情報タグを RevenueCat に送信することはありません。

アクセス許可

  • カメラ: 写真の撮影と動画の録画に必要です。
  • マイク: 音声付きで動画を録画するときにのみ必要です。
  • 位置情報: 写真の位置情報タグをオンにしたときにのみ使用します。
  • 写真ライブラリ: 撮影内容と書き出しをデバイスに保存するために必要です。
