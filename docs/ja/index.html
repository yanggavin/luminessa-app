サポート

Luminessa は、iPhone でのリアルタイム LUT プレビュー、写真撮影、動画録画、書き出しのために設計されています。

技術的なサポートが必要な場合は、こちらまでメールでお問い合わせください: support@luminessa-app.newtree.tech.

よくある質問

LUT はどのように適用しますか?

カメラ画面を開き、LUT ストリップをスワイプすると、シネマティックなカラースタイルをリアルタイムでプレビューできます。

写真や動画はどのように書き出しますか?

ギャラリーを開いて撮影内容をプレビューし、書き出し操作で写真ライブラリに保存します。

一部の機能が Pro と表示されるのはなぜですか?

Luminessa Pro はプレミアムなクリエイティブ操作をアンロックし、Apple の購入を通じて管理されます。サブスクリプションは Apple ID の設定から管理できます。
