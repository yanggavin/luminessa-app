Soporte

Luminessa está diseñada para previsualizar LUT en tiempo real, capturar fotos, grabar vídeo y exportar desde tu iPhone.

Para soporte técnico, escríbenos a support@luminessa-app.newtree.tech.

Preguntas frecuentes

¿Cómo aplico un LUT?

Abre la vista de cámara y desliza por la tira de LUT para previsualizar estilos de color cinematográfico en tiempo real.

¿Cómo exporto una foto o un vídeo?

Abre la galería, previsualiza la captura y usa la acción de exportar para guardarla en tu Fototeca.

¿Por qué algunas funciones aparecen como Pro?

Luminessa Pro desbloquea controles creativos premium y se gestiona mediante compras de Apple. Puedes administrar las suscripciones desde los ajustes de tu Apple ID.
