支持与帮助

光兮支持实时 LUT 预览、照片拍摄、视频录制和 iPhone 本地导出。

如果您需要技术支持，请发送邮件至 support@luminessa-app.newtree.tech。

常见问题

如何切换 LUT 预设？

打开相机界面，在 LUT 列表中滑动，即可实时预览不同的电影感色彩。

如何导出照片或视频？

打开图库，预览拍摄内容，然后使用导出操作保存到相册。

为什么有些功能显示为 Pro？

Luminessa Pro 可解锁高级创作控制，并通过 Apple 购买系统管理。您可以在 Apple ID 设置中管理订阅。
