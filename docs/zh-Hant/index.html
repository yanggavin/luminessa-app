支援與幫助

光兮支援即時 LUT 預覽、照片拍攝、影片錄製和 iPhone 本機匯出。

如果您需要技術支援，請寄送電子郵件至 support@luminessa-app.newtree.tech。

常見問題

如何切換 LUT 預設？

打開相機介面，在 LUT 列表中滑動，即可即時預覽不同的電影感色彩。

如何匯出照片或影片？

打開圖庫，預覽拍攝內容，然後使用匯出操作儲存到相簿。

為什麼有些功能顯示為 Pro？

Luminessa Pro 可解鎖進階創作控制，並透過 Apple 購買系統管理。您可以在 Apple ID 設定中管理訂閱。
